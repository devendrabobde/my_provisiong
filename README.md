onestop
=======
  
  
Requirement
===========

1. ruby 1.9.3p448
2. Rails 3.2.13

How to install application
1. Install RVM 
2. Install 'rvm install ruby 1.9.3p448'
3. copy cp config/database.yml.sample to config/database.yml
4. copy cp config/constants.rb.sample to config/constants.rb
5. bundle install
